---
name: housekeeping
description: Keep project docs and metadata in sync. Use when README, AGENTS.md, snapshots, gitignore, changelog, or project hygiene needs checking.
---

1. Check whether `README.md`, `AGENTS.md`, and `.codex/SNAPSHOT.md` match the
   current project state.
2. Check `.gitignore` once a stack creates generated files.
3. Avoid changelog or version edits unless the project has adopted those files.
4. Keep documentation short and operational.
5. Update only files whose drift is caused by the current task.
